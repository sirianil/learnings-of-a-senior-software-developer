# The Senior Engineer's Field Guide

I have compiled my learnings of working as a Senior Software Developer for more than 5 years in the industry. These documents contain my experiences and learnings of designing, developing and reviewing code, being on call, handling major incidents, running post-mortems and collaborating with other teams. I have worked in cross-functinal projects and on different teams.
This is designed as a reference for myself and for anyone else that may find it helpful. 
This is written with the help of Claude Code.
